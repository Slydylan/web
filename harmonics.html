Dimensionless Harmonic Ratios

The harmonic layer is now ratio-based. A frequency in hertz is an implementation choice for sound output, not a universal constant. The invariant object is the ratio.

Core representation

base phase tick: ω₀ = 1
harmonic n:      ω_n = nω₀
phase update:    θ ← wrap(θ + ω_n Δτ)
ratio:           ρ = ω_a/ω_b

What changed

The previous fixed-frequency story has been retired. Harmonics are useful as normalized phase ratios for search, visualization, audio rendering, and commit scoring. They are not evidence for a special physical resonance unless measured independently.
